# Calil Tarot Services

This is a web project developed with Next.js to present the services, biography, and contact information of Tarot Reader Carol Calil.

## Features

- "About Me" page with detailed biography
- Professional image gallery
- Introduction video
- Contact links (WhatsApp, Instagram, YouTube)

## Project Structure

- `app/` — Main site pages and layout
- `components/` — Reusable components
- `content/about.ts` — Biographical content and contact information
- `assets/` — Images used on the site
- `public/` — Public files (icons, SVG images)

## How to Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Go to `http://localhost:3000` in your browser.

## Useful Scripts

- `npm run dev` — Starts the development server
- `npm run build` — Builds the project for production
- `npm start` — Starts the server in production mode
